# User Stories

## Version 1

Définition des "user-stories" pour une première version fonctionnelle de mon application.

| **En tant que** | **Je souhaite pouvoir**        | **Dans le but de**                                            |
| --------------- | ------------------------------ | ------------------------------------------------------------- |
| Utilisateur     | Accéder à la page d'accueil    | Voir l'état actuel du kanban                                  |
| Utilisateur     | Créer une liste                | Ajouter une liste au kanban                                   |
| Utilisateur     | Modifier une liste             | Changer ses informations                                      |
| Utilisateur     | Supprimer une liste            | Enlever une liste de l'affichage et supprimer ses cartes      |
| Utilisateur     | Créer une carte                | Ajouter une carte à une liste                                 |
| Utilisateur     | Modifier une carte             | Mettre à jour une carte (contenu ou couleur)                  |
| Utilisateur     | Supprimer une carte            | Enlever une carte de sa liste                                 |
| Utilisateur     | Créer un label                 | Ajouter un label personnalisé                                 |
| Utilisateur     | Modifier un label              | Modifier son nom ou sa couleur                                |
| Utilisateur     | Supprimer un label             | Enlever un label et le retirer des cartes où il était assigné |
| Utilisateur     | Associer un label à une carte  | Qualifier une carte                                           |
| Utilisateur     | Dissocier un label d'une carte | Retirer la qualification de la carte                          |
